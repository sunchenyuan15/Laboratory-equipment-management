# 服务器jupyterlab

已经配置好service

使用如下命令可查看jupyterlab状态

```shell
systemctl status jupyterlab  # 查看状态
systemctl start jupyterlab   # 打开
systemctl stop jupyterlab    # 停止
systemctl enable jupyterlab  # 设置自启动
```
